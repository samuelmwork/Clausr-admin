# Clausr-admin

The administrative portal for Clausr, built with Next.js, Tailwind CSS, and Supabase.

## Features

- **Dashboard**: Real-time metrics for organizations, users, and contracts.
- **Organization Management**: Search, filter, and manage subscription plans and limits.
- **Alert System Monitor**: Monitor email delivery status with an automated 8 AM processing rule.
- **Data Export**: Export administrative data to CSV.

## Development

1. Install dependencies: `npm install`
2. Set up environment variables in `.env.local`
3. Run the development server: `npm run dev`

## Deployment

Deploy to Vercel by connecting this repository and configuring the environment variables.
